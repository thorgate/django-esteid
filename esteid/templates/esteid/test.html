Esteid Test page

This is a test page for digital signing of files via python.

  • Mobile ID signing is done with Demo Mobile ID service, these test numbers can be used. The signatures will not be valid unless the test certificates are used to validate them.
  • ID Card signing is done with real ID cards and as such, creates real signatures that can be validated with digidoc-tool.
  • ID Card signing requires the chrome-token-signing for chrome or firefox.
  • ID Card signing requires HTTPS which can be done with ngrok.
